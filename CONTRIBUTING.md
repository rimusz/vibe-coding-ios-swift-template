# Contributing to Vibe Coding iOS/Mac Swift Template

Thank you for your interest in contributing! This document provides guidelines for contributing to this project.

## Code of Conduct

- Be respectful and inclusive
- Welcome newcomers and help them learn
- Focus on constructive feedback
- Maintain a positive environment

## How to Contribute

### Reporting Issues

- Use the GitHub issue tracker
- Provide a clear description of the issue
- Include steps to reproduce if applicable
- Mention your environment (OS, Swift version, Xcode version)

### Submitting Changes

1. **Fork the Repository**
   ```bash
   git clone https://github.com/YOUR-USERNAME/vibe-coding-ios-swift-template.git
   ```

2. **Create a Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make Your Changes**
   - Follow the Swift API Design Guidelines
   - Add tests for new functionality
   - Update documentation as needed
   - Ensure all tests pass

4. **Commit Your Changes**
   ```bash
   git add .
   git commit -m "Add feature: description of your changes"
   ```

5. **Push to Your Fork**
   ```bash
   git push origin feature/your-feature-name
   ```

6. **Open a Pull Request**
   - Provide a clear description of the changes
   - Reference any related issues
   - Ensure CI checks pass

## Development Guidelines

### Code Style

- Follow Swift API Design Guidelines
- Use meaningful variable and function names
- Add comments for complex logic
- Keep functions focused and small

### Testing

- Write tests for all new features
- Ensure existing tests still pass
- Aim for high test coverage
- Use descriptive test names

### Documentation

- Update README.md for significant changes
- Add inline documentation for public APIs
- Include code examples where helpful

### Commit Messages

Use clear, descriptive commit messages:

```
Add feature: Brief description

Longer explanation of what changed and why.
Include any relevant context or decisions made.
```

## Questions?

Feel free to open an issue for questions or discussions about contributions.
